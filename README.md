# Three.js-2
Shooting canons
Commands:

- "4" wireframe or not

- "Q" select top cannon

- "W" select middle cannon

- "E" select bottom cannon

- "1" Top view

- "2" Perspective view of field

- "3" Moving Camera

- "left arrow" move selected canon

- "right arrow" move selected canon

- "SPACE BAR" shoot

- "R" add axis to balls
